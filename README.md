#geoVend

REAL-TIME GEOSPATIAL PLATFORM

geoVend is a smart street vendor system focused on real-time vendor tracking, geospatial mapping, and monitoring city zoning regulations.

For Consumers: Everyday users get an interface designed with minimal friction. They use a Live Discovery Map to see real-time pings of active vendors around them to quickly find local street food and goods.


For Street Vendors: Vendors access a mobile-friendly dashboard designed so it doesn't distract from their physical business. The hub includes a massive "Go Online / Go Offline" toggle to control their real-time location tracking. The system is also IoT-enabled, actively warning vendors if their cart drifts into restricted "No Vending" zones.


For City Planners (Admins): Authorities utilize a data-dense, desktop-first command center. This portal features a "god-mode" Global Geospatial Map that overlays vendor locations with green (permitted) and red (no-vending) zoning boundaries to spot bottlenecks or illegal clustering. It also includes an Urban Planning Analytics Hub that uses historical data, heatmaps of foot traffic, and vendor density to help planners make informed decisions.
